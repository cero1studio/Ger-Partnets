# Portal de Aliados Estratégicos – Global Express

Plataforma web para la gestión de leads y referidos del programa de reclutamiento internacional de **Global Express**. Desarrollada por el **Área Digital de Global Express**.

## Tecnologías

- [Next.js](https://nextjs.org) (App Router)
- [Tailwind CSS](https://tailwindcss.com)
- [shadcn/ui](https://ui.shadcn.com)
- [Vercel Analytics](https://vercel.com/analytics)

## Desarrollo local

```bash
pnpm dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Deploy

Cada merge a `main` despliega automáticamente a producción.
